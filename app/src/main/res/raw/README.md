# res/raw — Ses Dosyaları

Bu klasör ilahi ve zikir ses dosyaları için ayrılmıştır.

## Gerekli dosyalar

| Dosya Adı | Açıklama |
|-----------|----------|
| `zikir_tik.ogg` | Zikir sayaç sesi (hafif tıklama) |
| `zikir_tamam.ogg` | Tur tamamlanma sesi |
| `sala.ogg` | Sala/salat sesi |

## Kaynak Önerileri

- Ücretsiz CC0 ses efektleri: https://freesound.org
- Dosyaları OGG formatına dönüştürün (Android tercihi)
- Her dosyayı bu klasöre doğrudan kopyalayın
- Dosya adı lowercase + alt çizgi kullanın (ör: `zikir_tik.ogg`)
